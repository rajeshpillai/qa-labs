This page is the baseline for kata 58. It's deliberately well-built so your Web Vitals come back green:

FCP — fires almost immediately because nothing render-blocks the first paint.
LCP — the .hero gradient block is the largest above-the-fold element. It paints inline with first paint, so LCP ≈ FCP.
CLS — every element on this page has a fixed size declared in CSS. Nothing reflows after load.
INP — there's no JavaScript, so input handlers have no work to do.
