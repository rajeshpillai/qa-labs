KATA 40

Test Data and Fixtures

About This Kata

This kata teaches you how to manage test data using JSON fixture files. Instead of hard-coding data in every test, you store it in a shared file and load it at test time.

The tests in this kata target the KYC Onboarding Form from Kata 26.

Open Kata 26: KYC Onboarding Form
